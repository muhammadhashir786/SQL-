ADD Constraints In Table That Already Created:

1. to add a UNIQUE constraint in a table that already created:

Syntax: ALTER TABLE table_name ADD CONSTRAINT constraint_name UNIQUE (column_name);

Query: 	ALTER TABLE users ADD CONSTRAINT unique_email UNIQUE (email);

Note: at "column_name" type the column name where you want to apply constraint.

HERE WE CHANGED IN THE STRUCTURE/TABLE/SCHEMA SO WE USE MODIFY COLUMN:

2. to change any column to NOT NULL:

Syntax: ALTER TABLE table_name MODIFY COLUMN column_name datatype NOT NULL;

Query: ALTER TABLE users MODIFY COLUMN name VARCHAR(100) NOT NULL;

3. to change any column to NULL:

Syntax: ALTER TABLE table_name MODIFY COLUMN column_name datatype NULL;

Query: ALTER TABLE users MODIFY COLUMN name VARCHAR(100) NULL;

4. to add CHECK constraint in the column of table:

Syntax: ALTER TABLE table_name ADD CONSTRAINT constraint_name CHECK (column_name define_condition);

Query: ALTER TABLE users ADD CONSTRAINT chk_dob CHECK (date_of_birth > '1970-01-01');
