Insert Data Into Table:

1. to insert 1 row/full row into table:	OR 
   to insert 1 row without specify column names:

Syntax:	INSERT INTO table_name VALUES
	(insert data here);

Query:	INSERT INTO users VALUES
	(1, 'Ahmad', 'ahmad@gmail.com', 'Male', '2000-01-01', DEFAULT);

2. to insert data into specific columns in 1 row of the table:
   to insert 1 row with specify column names:

	type the "names of the column" in which you want to insert data in the below Query:
	
Syntax:	INSERT INTO table_name (1st_column_name, 2nd_column_name, ...) VALUES
	(1st_column_data, 2nd_column_data, ...);

Query:	INSERT INTO users (name, email, gender, date_of_birth) VALUES
	('Ahmad', 'ahmad@example.com', 'Male', '2000-01-01');

Here we can skip columns and also change the order of the columns when inserting data into table,
but data is insert into the columns orderwise automatically like below example.

Query:	INSERT INTO users (email, name, date_of_birth, gender) VALUES
	('ahmad@gmail.com', 'Ahmad', '2000-01-01', 'Male');

3. to insert data into multiple rows in specific columns at once (recommended):
   to insert multiple rows with specify column names:

Syntax:	INSERT INTO table_name (1st_column_name, 2nd_column_name, ...) VALUES
	(insert_data_into_1st_row),
	(insert_data_into_2nd_row),
	(insert_data_into_3rd_row);

Query:	INSERT INTO users (name, email, gender, date_of_birth) VALUES
	('Imran', 'imran@example.com', 'Male', '2002-03-24'),
	('Kashif', 'kashif@example.com', 'Male', '2005-05-15'),
	('Bilal', 'bilal@example.com', 'Male', '2003-01-21'); 

Note: The remaining columns like id which is AUTO_INCREMENT and
      created_at (which has a default value set) are automatically handled by MySQL.
